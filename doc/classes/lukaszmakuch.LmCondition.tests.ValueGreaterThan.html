isNegated()

isNegated() : boolean

Returns true if this condition is negated.

If this condition is already negated, then calling this method takes no effect. It doesn't change result of any method desribing the condition, so if there's a "value greater than X" condition, then it becomes "negated (value greater than X)", NOT "value lower or equal X".

boolean

negate()

negate() : null

Negates this condition.

Calling this method two times doesn't restore the previous state. It means that if it is a "value greater than X" condition, than calling this method makes it "value NOT greater than X" and calling it again takes no effect as this stays "value NOT greater than X".
