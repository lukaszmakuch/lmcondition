compose()

compose(array  $decomposedComposite) : \lukaszmakuch\LmCondition\ConditionComposite

Rows of given array 2D represents OR conditions while columns represents AND conditions.

Array of Condition objects like this:

[
    [A, B],
    [C, D]
]

is composed into ConditionComposite like this:

((A AND B) OR (C AND D))
